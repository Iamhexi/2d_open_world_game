# 2d_open_world_game
It's the 2D open world game written in C++20 using SFML library for graphics. Work in progress.

## Requirements
Software required to build the game:

- make

- cmake (version >= 3.22)

- sfml (version >= 2.6)

- gtest

## Installation
`cmake . -B build`
`cd build`
`make`
`./run`

## Gameplay

### Key bindings

| **Key** | **Action** |
|-------|-------|
| Enter/Return | Go to the next dialogue line during a dialogue. |
| f | Pick up an item. |
| w | Move up. |
| s | Move down. |
| a | Move left. |
| d | Move right. |
